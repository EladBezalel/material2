md-button is an HTML <button> or <a> tag enhanced with styling and animation to match the spec of the Material Design button spec

Theming

All button types can be themed to match you "primary" palette, your "accent" palette, or your "warn" palette using the color attribute. Simply pass in the template name.

In flat button the palette chosen will affect the text color, while in other buttons, it affects the background.

Accessibility

In high contrast mode, a strong border is added to the button to make it easier to see.

Button focus events originating from the keyboard wil retain focus styles, while button focus events from mouse will not.

As md-button is added to an existing button or tag, it enjoys all the accessibility natively built into these elements.
